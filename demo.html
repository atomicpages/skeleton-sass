Foreword

This is a demo showcasing the default styles of the skeleton css framework and the functionality of the media queries. I encourage you resize your browser window or view this file on a smart phone.

Also, a special note on the form elements at the bottom of the page. These utilize new HTML 5 form types such as tel, search, url, and email. I highly recommend viewing these new elements on a mobile device to see how easy they make life for mobile users! Seriously, you'll be amazed.
