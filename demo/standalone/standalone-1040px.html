Foreword

This is a demo showcasing the default styles of the Skeleton Sass and the functionality of the media queries. Resize your browser window or view this file on a smart phone and see the goodness.

The Code

Simply change the value in _MYconfig.scss or the name of your global configuration file to the code below:

$base-col-width: 45px; // 12 * (45 + 20) = 1040px
// $base-width is calculated automatically for you but it can be overridden easily
